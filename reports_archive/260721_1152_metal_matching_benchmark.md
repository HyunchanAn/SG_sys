# Step 2 금속 피착재 매칭(Material ID Determination) 벤치마크 보고서

## 1. 개요
본 보고서는 Step 1에서 계측된 금속 피착재의 물리적 특성(표면에너지, 조도, 광택도)을 활용하여 Step 2에서 금속의 재질 및 마감(Finish) 종류를 정확히 판별해내는지 300회 반복 테스트한 결과입니다.

## 2. 테스트 환경 및 방법
- **데이터 원본**: `postgresql://localhost:5432/sg_proj_004` 실제 운영 DB 내 `adherend_properties` 테이블
- **테스트 횟수**: 300회 무작위 샘플링 복원 추출
- **노이즈 환경**: 실제 계측 환경의 오차를 모사하기 위해, 각 속성(Ra, Gloss, SFE)의 실제값 대비 정규분포(Gaussian) 표준편차 5.0% 수준의 노이즈 삽입

## 3. 사용된 가중치 모델 및 종속 모듈
- **종속 모듈**: `SG_proj_010` (금속 재질 및 마감 판별 모델)
- **가중치 모델 여부**: 현재 `SG_proj_010` 모듈은 딥러닝 가중치(.pt, .onnx 등) 모델을 통한 추론 대신, `SubstrateMatcher` 클래스 내에 하드코딩된 **물리 형상 가중치 유클리디안 거리 기반 매칭(Euclidean Distance Matcher)** 알고리즘을 사용하고 있습니다.
  - 조도(Ra) 가중치: 45%
  - 광택도(Gloss) 가중치: 45%
  - 표면에너지(SFE) 가중치: 10%
- *(참고)* `README.md`에는 EfficientNet-V2 등 비전 AI 아키텍처가 명시되어 있으나, 실질적인 코드(`matcher.py`)는 수치형 데이터 기반의 매칭 방식을 채택하고 있습니다.

## 4. 벤치마크 성적 (평가 결과)
- **총 테스트 건수**: 300건
- **정답 건수**: 300건
- **정확도 (Accuracy)**: **100.00%**

PostgreSQL DB 기반 실 데이터를 활용하여 5%의 가우시안 노이즈가 유입된 상황에서도 위와 같은 정확도로 대체 피착재 매칭을 수행할 수 있음을 재확인했습니다.
