# Step 2 기성 제품 추천(TOPSIS/AHP Recommender) 벤치마크 보고서

## 1. 개요
본 보고서는 금속 표면에 자사의 어떤 기성 제품이 적합한지 추천해 주는 Step 2 모델(`SG_proj_012`)의 벤치마크 결과입니다. 실제 PostgreSQL 운영 DB에서 추출한 정답지-문제지 쌍을 기반으로 300회 반복 테스트하여 추천 성능을 검증했습니다.

## 2. 테스트 환경 및 방법
- **데이터 원본**: `postgresql://localhost:5432/sg_proj_004` 실제 운영 DB 내 `our_products` 테이블
- **테스트 횟수**: 300회
- **가정 사항**: 요구되는 유지력(Processability Level)은 가능한 한 높은 수준(Level 5)으로 고정하여 테스트
- **노이즈 환경**: 실제 계측기 오차를 감안해 표면에너지와 조도에 5% 수준의 정규분포(Gaussian) 노이즈 추가

## 3. 사용된 가중치 모델 및 종속 모듈
- **종속 모듈**: `SG_proj_012` (기성 제품 추천 모듈)
- **가중치 모델 여부**: 별도의 딥러닝/머신러닝 가중치 파일(`.pt`, `.pkl`)은 포함되어 있지 않으며, `matcher.py` 내의 **다기준 의사결정(MCDA/AHP)** 수식 가중치를 기반으로 추천 스코어를 산출합니다.
  - 표면에너지 오차 (40% 가중치)
  - 조도 오차 (20% 가중치)
  - 마감(Finish) 일치 여부 (20% 가중치)
  - 가공성(Processability) 레벨 패널티 (20% 가중치, 요구 조건 미달 시 Hard Fail 처리)

## 4. 벤치마크 성적 (평가 결과)
- **Top-1 정확도**: 89.00%
- **Top-3 정확도**: 100.00%
- **Top-5 정확도**: 100.00%

## 5. 비고 (경향성 분석)
- 실제 DB의 데이터를 테스트한 결과, **노이즈 유입 시 1순위 제품을 놓치는 오류**가 일부 발견되었습니다.
- **분석 내역**: SFE(표면에너지)의 경우 제품 간 목표값 격차가 조도(Roughness) 대비 적은 편인데 가중치가 40%로 가장 높게 설정되어 있습니다. 이로 인해 5% 노이즈가 SFE에 영향을 주었을 때, 순위가 쉽게 뒤바뀌는 현상(민감도 증가)이 나타납니다.
- **대응책 제언**: 실무 적용 시 노이즈에 대한 강건성을 확보하기 위해 SFE의 가중치를 약간 하향 조정하거나 오차를 비선형적(제곱 등)으로 패널티화하는 방식 도입을 검토할 필요가 있습니다.
